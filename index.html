Subjects in the First Year

PRA: Practical Work

You work independently on practical assignments in our practical classroom. Since you have to work independently, you can't always ask your questions immediately. Write them down to discuss them during the PRO lessons.

PRO: Practical Support

You receive coaching and support for practical assignments. There are also lessons on modules related to the V-model. These usually do not involve programming but focus on other aspects of our field.

WIN: Programming for Windows

You learn to build applications that run on a Windows computer. In this block, we work with the 'Python' language. To learn this language, we start with programming on a smaller device: the micro:bit. For more information, consult the module booklets.

WEB: Programming for the Web

This subject is about creating websites and web applications. We start with the basic markup languages HTML and CSS, which are the building blocks of a website. Afterward, you will learn to build an interactive website. For more information, check the module booklets.

KZD: Elective Digital Skills

Your first elective cannot be chosen; everyone follows 'digital skills' in this block. This will teach you everything you need to create neat reports and work smartly with your computer. However, you can choose your other three electives! This happens in blocks D and BPV-B.

COD: CodeLab

In CodeLab, there is no new teaching material. You can focus on homework, get extra explanations, and attend extra lessons. If your homework is done, and you understand everything, we challenge you to further develop yourself as a developer. Our instructors are available for extra help.

SLB: Study Career Guidance

During the class-based SLB hours, you learn everything about the program itself. What to pay attention to, how the program is structured, how to study best, and so on. You also work on career orientation modules.

Schedule for the First Year

The schedule for a first-year student varies by class, but typically, the timetable consists of approximately 22 class hours per week.

Block Schedule

Welcome to our four-year Software Developer program! We understand that understanding the program's structure is important, so we'd like to inform you about the block schedule. In the regular four-year variant, you will go through various blocks that will prepare you for a successful career in software development. These blocks are:

Regular 4-year

  • Block A: Smart City
  • Block B: Developerland
  • Block C: 4S + Internship Preparation
  • Block D: Orientation Internship
  • Block E: Specialization
  • Block F: Exam Block
  • Block G: Final Internship

Accelerated 3-year

  • Block A
  • Block BA
  • Block CA

  • Block GA
  • Block FA
